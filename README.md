# Minecrafter5

Minecrafter5 is a sandbox block-building game inspired by Minecraft.

## Features (Planned)
- Infinite world generation  
- Multiple dimensions (Overworld, Nether, End, Aether, Moon)  
- Survival and Creative modes  
- Crafting and inventory system  
- Multiplayer with invite codes  
- Mod support  
- Touchscreen controls (mobile)  
- Bluetooth controller support  

## Controls
- Touchscreen: Move, look, place, and break blocks  
- Controller: Full gameplay support  
- Keyboard (PC):  
  - WASD: Move  
  - Mouse: Look around  
  - Left Click: Break block  
  - Right Click: Place block
